# PyData Analysis Workshop

A hands-on exploration of data analysis techniques in Python using Jupyter Notebooks, based on the "Simplifying Data Analysis" workshop.

## Overview

This repository works through practical data analysis workflows using Python's core data stack. It covers data loading and cleaning, exploratory analysis, and visualisation techniques applicable to real-world datasets.

## Topics Covered

- Data loading and preprocessing with Pandas
- Exploratory data analysis (EDA)
- Visualisation with Matplotlib and Seaborn
- Working with Jupyter Notebooks interactively

## Getting Started

### Prerequisites
- Python 3.8+
- Jupyter Notebook or JupyterLab

### Setup
```bash
git clone https://github.com/marz1307/pydata-analysis-workshop
cd pydata-analysis-workshop
pip install -r requirements.txt
jupyter notebook
```
